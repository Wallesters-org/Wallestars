### Основна информация:

- **Име на хранилището**: Wallestars
- **Организация**: Wallesters-org
- **Адрес**: https://github.com/Wallesters-org/Wallestars
- **Тип**: Публично хранилище (public repository)

### Какво е GitHub хранилище?

GitHub хранилището е място за съхранение на код, документация и история на промените в проект. То използва системата за контрол на версиите Git и предоставя:

1. **Контрол на версиите** - проследяване на всички промени в кода
2. **Сътрудничество** - възможност за работа на множество разработчици
3. **Клонове (Branches)** - работа по различни функционалности едновременно
4. **Pull Requests** - преглед и одобрение на промени преди вливането им
5. **Issues** - проследяване на задачи и проблеми
6. **Документация** - чрез README файлове и Wiki

### Как да работите с това хранилище:

1. **Клониране** на хранилището:
   ```bash
   git clone https://github.com/Wallesters-org/Wallestars.git
   ```

2. **Създаване на нов клон** (branch):
   ```bash
   git checkout -b име-на-новия-клон
   ```

3. **Правене на промени** и commit:
   ```bash
   git add .
   git commit -m "Описание на промените"
   ```

4. **Изпращане на промените**:
   ```bash
   git push origin име-на-клона
   ```
